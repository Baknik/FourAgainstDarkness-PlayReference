
When asked to make a wandering enemy roll, there is a `1-in-6` chance that one appears.

When a wandering enemy appears, roll on the table below to determine it's type.

| `d6`  | Enemy Type |
| :---: | ---------- |
| `1-2` | [[Vermin]] |
| `3-4` | [[Minion]] |
|  `5`  | [[Weird]]  |
|  `6`  | [[Boss]]   |
