A settlement is a hub of civilization that contains many [[Building]]s. Each town has a size and an identity. When finding a new settlement, roll on the tables below to determine it's properties.

##### Size
Determines how many identities the settlement has, and how often random events happen.

| **`d6`** | **Size**    |
| :------: | ----------- |
|   1-3    | [[Village]] |
|   4-5    | [[Town]]    |
|    6     | [[City]]    |
##### Identity
Each identity comes with a group of thematically appropriate [[Building]]s.

| **`d6`** | **Identity**   |
| :------: | -------------- |
|    1     | [[Commercial]] |
|    2     | [[Lawless]]    |
|    3     | [[Spiritual]]  |
|    4     | [[Arcane]]     |
|    5     | [[Industrial]] |
|    6     | [[Frontier]]   |
