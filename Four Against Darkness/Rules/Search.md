When searching, roll on the following table.

| `d6`  | Search Result                                                    |
| :---: | ---------------------------------------------------------------- |
|  `1`  | A [[Wandering Enemy]] [[Surprise]]s the party                    |
| `2-4` | Nothing is found                                                 |
| `5-6` | Something has been found! Roll on the [[#Search Findings]] table |
#### Search Findings
| `d6`  | Search Findings       |
| :---: | --------------------- |
| `1-2` | A [[Clue]]            |
| `3-4` | A [[Hidden Treasure]] |
| `5-6` | A [[Secret Door]]     |
